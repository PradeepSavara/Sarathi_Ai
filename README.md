# Sarathi AI - Delivery Governance & KPI Dashboard

A comprehensive AI-powered platform for monitoring and optimizing delivery governance, KPIs, project health, team performance, and actionable insights across your organization.

## 📋 Project Overview

Sarathi AI is a full-stack application designed to provide real-time visibility into project delivery, team performance, and risk management. The platform leverages AI to generate intelligent insights and recommendations for improving delivery excellence.

## 🏗️ Project Structure

```
Sarathi_Ai/

│   ├── src/
│   ├── public/

│   ├── tailwind.config.js
│   └── README.md           # Frontend documentation
└── [Backend] (to be added)   # Backend API service
```

## 🚀 Quick Start

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173) to access the application.

## 🛠️ Tech Stack

### Frontend
- React 18+ with TypeScript
- Vite for fast builds
- Tailwind CSS for styling
- React Context API for state management
- Recharts for data visualization

### Backend (To be configured)
- Node.js/Express or alternative
- Database (MongoDB, PostgreSQL, etc.)

## 📝 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Sarathi_Ai
   ```

2. **Set up Frontend**
   ```bash
   cd Frontend
   npm install
   npm run dev
   ```

3. **Set up Backend** (when available)
   ```bash
   cd Backend
   npm install
   npm start
   ```
